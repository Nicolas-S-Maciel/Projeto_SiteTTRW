## 📝 Sobre o Projeto
Este é um **Hub de Conteúdo** desenvolvido para centralizar redes sociais, lives e contatos comerciais. O projeto foca em uma interface de alta fidelidade visual (Cyberpunk Aesthetics) e performance otimizada.

## 🛠️ Tecnologias e Ferramentas
* **Frontend:** HTML5 e CSS3 (Design Responsivo & Media Queries).
* **Versionamento:** Git & GitHub.
* **Deploy & CI/CD:** Vercel (Atualização automática via Push).
* **Backend as a Service:** Formspree (Gestão de formulários e segurança).

## 📱 Responsividade
O projeto foi desenvolvido seguindo princípios de **Mobile-First**, garantindo que a experiência do usuário seja fluida tanto em desktops quanto em smartphones, ajustando grids e elementos neon dinamicamente.

1.  O site está hospedado em: [https://site-niko-ttrw.vercel.app/](https://site-niko-ttrw.vercel.app/)
